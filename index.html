Regex Patterns Used

  • /^\S(?:.*\S)?$/ - Description validation (no leading/trailing spaces)
  • /^(0|[1-9]\d*)(\.\d{1,2})?$/ - Amount validation
  • /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[12]\d|3[01])$/ - Date format (YYYY-MM-DD)
  • /^[A-Za-z]+(?:[ -][A-Za-z]+)*$/ - Category validation
  • /\b(\w+)\s+\1\b/ - Duplicate word detection (advanced)
